Principais formatações de texto

Negrito/ Destaque

Neste texto temos um termo em negrito usando a tag <b> (não semântica).

Neste paragráfo temos um termo em destaque usando a tag <strong> (com semântica).

Itálico/ Ênfase

Neste texto temos um termo em Itálico usando a tag <i> (não semântica).

Neste texto temos um termo em Ênfase usando a tag <em> (com semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

E no outro texto temos outra marcação com a tag <mark>.

Texto Grande e texto pequeno

Estamos criando um texto Grande e criando um texto pequeno com a tag <big> e a tag <small>.

Texto excluído

Podemos marcar um texto como excluído para dizer que ele deve ser lido mas desconsiderado. com a tag <del>.

Texto inserido

Podemos marcar também um texto como inserido pra dizer que ele foi adicionado depois com a tag <ins>.

Existe também o sublinhado com a tag <u>. (não semântica.)

Texto sobrescrito

Para inserir coisas do tipo x30+3, com a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H2O com a tag <sub>
